# 🧪 בדיקה מהירה - נושא מרכזי

## ✅ בדיקות לביצוע:

### 1. תצוגה מקדימה
```
□ התמונה מוצגת במלואה (לא חתוכה)
□ יש רקע כהה סביב התמונה
□ הכותרת והתיאור קריאים
□ כפתור ה-play אדום ויפה
```

### 2. כפתור עריכה (אדמין בלבד)
```
□ כפתור העיפרון מופיע בפינה השמאלית
□ לחיצה עליו פותחת את פאנל האדמין
□ נפתח בטאב "נושא מרכזי"
```

### 3. נגן YouTube
```
□ לחיצה על הכרטיס פותחת modal
□ הסרטון מתחיל להתנגן אוטומטית
□ יש כפתור X לסגירה
□ יש כפתור YouTube לפתיחה באפליקציה
□ הסאונד עובד
□ מסך מלא עובד
```

### 4. כללי
```
□ הכרטיס מופיע בראש מסך הבית
□ אין שגיאות בקונסול
□ האפליקציה לא קורסת
□ הכל חלק ומהיר
```

```

```

```

```
