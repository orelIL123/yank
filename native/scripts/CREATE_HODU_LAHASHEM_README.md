# יצירת טבלת הודו לה' (סיפורי ניסים)

## 📋 הוראות

### שלב 1: הרצת SQL Script

1. פתח את Supabase Dashboard: https://app.supabase.com
2. בחר את הפרויקט שלך
3. לך ל-SQL Editor (בתפריט השמאלי)
4. לחץ על "New Query"
5. העתק את כל התוכן מהקובץ: `create-hodu-la-hashem-table.sql`
6. הדבק ב-SQL Editor
7. לחץ על "Run" (או Ctrl+Enter)

### שלב 2: בדיקה

לאחר ההרצה, אמור לראות הודעה:
```
hodu_la_hashem table created successfully! ✨
```

### שלב 3: שימוש

עכשיו תוכל:
- להוסיף סיפורי ניסים דרך מסך האדמין → טאב "הודו לה'"
- לערוך סיפורים קיימים
- למחוק סיפורים
- כל הסיפורים יופיעו במסך "הודו לה'" באפליקציה

## 📝 מבנה הטבלה

הטבלה `hodu_la_hashem` מכילה:
- `id` - מזהה ייחודי
- `data` - JSONB עם כל הנתונים:
  - `title` - כותרת הסיפור
  - `content` - תוכן הסיפור
  - `isActive` - האם הסיפור פעיל
  - `createdAt` - תאריך יצירה
- `created_at` - תאריך יצירה (PostgreSQL timestamp)
- `updated_at` - תאריך עדכון אחרון

## 🔒 אבטחה

- קריאה: כל המשתמשים יכולים לקרוא
- כתיבה/עריכה/מחיקה: רק משתמשים מאומתים (authenticated)
